내용을 작성할 때에는 강조를 하고자 할 때가 있다.
굵은 글씨를 사용하거나, 기울이거나, 밑줄을 표기하거나, 문장완성 취소선을 넣는다.
이러한 기능이 있지만, 위 내용은 모두 강조의 의미가 아니다.

웹문서에서 강조의 의미는 뜻 전달이 명확하게 가능해야 한다.
즉, 중요한 강조를 뜻하는 위험과 유사한 코드
중요한 의미이지만, 강요보다 권유의 성격을 띄고 있는 코드
강조 처리되는 기능으로 재확인의 의미, 기억의 의미

i요소는 감정의 표현이라는 의미를 가지므로 주로 아이콘 또는 이모티콘 용도로 사용. s요소(srtike)는 html5에서 완전히 삭제되어진 요소이므로 더 이상 사용할 수 없다.
